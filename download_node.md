# Node.js Portable 다운로드 가이드

## Windows용 Node.js 다운로드

1. 다운로드 링크: https://nodejs.org/dist/v22.22.0/node-v22.22.0-win-x64.zip

2. 압축 해제:
   - `src-tauri/resources/node-portable/` 폴더에 압축 해제
   - 폴더 구조:
     ```
     src-tauri/resources/node-portable/
     ├── node.exe
     ├── npm
     ├── npm.cmd
     └── ... (기타 파일들)
     ```

3. 확인:
   - `node.exe`와 `npm.cmd`가 존재하는지 확인

## Linux/macOS용 Node.js 다운로드

Linux: https://nodejs.org/dist/v22.22.0/node-v22.22.0-linux-x64.tar.gz
macOS: https://nodejs.org/dist/v22.22.0/node-v22.22.0-darwin-x64.tar.gz

압축 해제 후 동일한 위치에 배치
