# moldClaw 이용약관 및 면책조항

**시행일:** 2026년 2월 24일
**버전:** 1.0
**적용 법률:** 대한민국 법률

---

## 제1조 (목적)

이 약관은 moldClaw(이하 "본 소프트웨어")의 설치 및 사용에 관한 기본적인 사항과 사용자의 권리·의무, 책임사항 및 면책조항을 규정함을 목적으로 합니다.

본 소프트웨어는 OpenClaw 오픈소스 프로젝트의 데스크톱 인터페이스로서, MIT 라이선스에 따라 배포됩니다.

---

## 제2조 (정의)

1. **"본 소프트웨어"**란 moldClaw 데스크톱 애플리케이션 및 관련 구성요소를 말합니다.
2. **"OpenClaw"**란 본 소프트웨어가 의존하는 오픈소스 AI 에이전트 플랫폼을 말합니다.
3. **"사용자"**란 본 소프트웨어를 설치하거나 사용하는 개인 또는 법인을 말합니다.
4. **"AI 서비스"**란 Anthropic Claude, OpenAI GPT 등 본 소프트웨어를 통해 연결되는 제3자 AI 모델 서비스를 말합니다.
5. **"메신저 서비스"**란 Telegram, Discord, WhatsApp, Slack 등 본 소프트웨어를 통해 연결되는 제3자 메시징 플랫폼을 말합니다.

---

## 제3조 (이용 동의)

사용자가 본 소프트웨어를 설치하거나 사용하는 경우, 이 약관의 모든 조항에 동의한 것으로 간주합니다.

---

## 제4조 (소프트웨어 특성 고지)

### 4.1 오픈소스 소프트웨어

본 소프트웨어는 OpenClaw 오픈소스 프로젝트(MIT License, Copyright © 2025 Peter Steinberger)를 기반으로 합니다. 원본 소프트웨어의 모든 제한사항과 면책조항이 본 소프트웨어에도 동일하게 적용됩니다.

### 4.2 AI 에이전트 시스템

본 소프트웨어는 AI 모델이 사용자를 대신하여 작업을 수행하는 "에이전트" 시스템입니다. AI의 판단과 행동은 예측 불가능할 수 있으며, 사용자가 의도하지 않은 결과를 초래할 수 있습니다.

### 4.3 시스템 접근 권한

본 소프트웨어는 다음 시스템 자원에 접근할 수 있습니다:
- 파일 시스템 (읽기/쓰기)
- 셸 명령 실행
- 네트워크 통신
- 브라우저 자동화
- 클립보드

## 제6조 (대한민국 법률 관련 고지)

### 6.1 개인정보 보호법 관련

#### 6.1.1 개인정보 수집·이용 고지

본 소프트웨어는 다음 개인정보를 **사용자의 로컬 컴퓨터**에만 저장합니다:

| 수집 항목 | 수집 목적 | 보유 기간 | 저장 위치 |
|----------|----------|----------|----------|
| AI 서비스 API 키 | AI 모델 호출 | 사용자 삭제 시까지 | ~/.openclaw/openclaw.json |
| 메신저 봇 토큰 | 메신저 연동 | 사용자 삭제 시까지 | ~/.openclaw/openclaw.json |
| 외부 서비스 API 키 | 도구/스킬 연동 | 사용자 삭제 시까지 | ~/.openclaw/openclaw.json |
| 대화 기록 | 세션 관리 | 사용자 삭제 시까지 | ~/.openclaw/sessions/ |

**중요**: 본 소프트웨어는 수집한 정보를 외부 서버로 전송하지 않습니다. 단, 사용자가 연결한 제3자 서비스(AI 제공업체, 메신저 플랫폼 등)로는 해당 서비스 이용에 필요한 데이터가 전송됩니다.

#### 6.1.2 제3자 제공 관련

본 소프트웨어를 통해 다음 제3자에게 데이터가 전송될 수 있습니다:

| 제3자 | 전송 데이터 | 목적 | 법적 근거 |
|-------|-----------|------|----------|
| AI 서비스 제공자 (Anthropic, OpenAI 등) | 사용자 입력, 대화 내용 | AI 응답 생성 | 사용자 동의 (API 키 입력) |
| 메신저 플랫폼 (Telegram, Discord 등) | 메시지 내용 | 메시지 송수신 | 사용자 동의 (연동 설정) |
| 웹사이트 (web_fetch, 브라우저 도구) | HTTP 요청 | 웹 콘텐츠 추출 | 사용자 지시 |

**개인정보 보호법 제17조(개인정보의 제공)** 관련: 사용자가 본 소프트웨어에 API 키를 입력하고 서비스를 연동하는 행위는 해당 제3자에 대한 개인정보 제공 동의로 간주됩니다.

#### 6.1.3 그룹 채팅 관련 주의사항

**개인정보 보호법 제15조(개인정보의 수집·이용)** 관련:

그룹 채팅에서 본 소프트웨어를 사용하는 경우, 그룹 내 **제3자의 메시지**가 AI 모델로 전송될 수 있습니다. 사용자는 다음 의무가 있습니다:

1. 그룹 참여자에게 AI 봇 사용 사실을 고지할 것
2. 또는 `groupPolicy: "allowlist"` 설정으로 처리 범위를 제한할 것

제3자 동의 없이 그 메시지를 AI에게 전송하는 행위는 개인정보 보호법 위반에 해당할 수 있습니다.

### 6.2 정보통신망법 관련

#### 6.2.1 정보통신망 침해행위 금지 (제48조)

본 소프트웨어의 다음 기능 사용 시 정보통신망법 위반 가능성이 있습니다:

| 기능 | 위험 행위 | 관련 조항 |
|------|----------|----------|
| `exec` 도구 | 타인 시스템 무단 접근 명령 | 제48조 제1항 (정보통신망 침입 금지) |
| `browser` 도구 | 웹사이트 자동화 ToS 위반 | 제48조 제2항 (정보통신망 장애 유발 금지) |
| `web_fetch` 도구 | 과도한 요청으로 서비스 방해 | 제48조 제3항 (악성프로그램 전달 금지) |

**사용자 책임**: 사용자는 본 소프트웨어를 이용하여 타인의 정보통신망에 무단으로 침입하거나 장애를 유발하는 행위를 해서는 안 됩니다.

#### 6.2.2 개인정보 누설 금지 (제49조)

본 소프트웨어가 수집하거나 처리하는 정보를 제3자에게 누설하거나 도용하는 행위는 정보통신망법 제49조 위반에 해당합니다.

### 6.3 저작권법 관련

#### 6.3.1 저작물 무단 복제 (제136조)

`web_fetch`, `browser` 도구를 이용한 웹사이트 콘텐츠 추출은 해당 웹사이트의 저작권 및 이용약관을 준수해야 합니다.

- 저작물의 공정 이용(제35조의5) 범위 내에서만 사용할 것
- 상업적 목적 사용 시 저작권자 허락을 받을 것
- robots.txt 및 이용약관을 확인할 것

### 6.4 형법 관련

#### 6.4.1 컴퓨터등 사용사기 (제347조의2)

본 소프트웨어를 이용하여 컴퓨터 등 정보처리장치에 허위의 정보 또는 부정한 명령을 입력하여 재산상의 이익을 취득하는 행위는 형법 제347조의2(컴퓨터등 사용사기)에 해당합니다.

#### 6.4.2 컴퓨터등 장애 업무방해 (제314조 제2항)

본 소프트웨어를 이용하여 컴퓨터 등 정보처리장치 또는 전자기록 등 특수매체기록을 손괴하거나 정보처리장치에 허위의 정보 또는 부정한 명령을 입력하여 타인의 업무를 방해하는 행위는 형법 제314조 제2항에 해당합니다.

### 6.5 전자상거래법 관련

#### 6.5.1 부정한 방법의 거래행위

`browser` 도구를 이용한 자동 구매, 예매, 선착순 이벤트 참여 등은 해당 사이트의 이용약관 위반일 수 있으며, 경우에 따라 전자상거래법 위반에 해당할 수 있습니다.

## 제7조 (특수 서비스 관련 고지)

### 7.1 WhatsApp 연동 관련

**⚠️ 중요 경고**

WhatsApp 연동은 **비공식 API (whatsmeow 라이브러리)**를 사용합니다. 이는 Meta(WhatsApp)의 이용약관을 위반하는 행위입니다.

**Meta 이용약관 위반 내용:**
> "You may not access, reproduce, download, distribute, transmit, broadcast, display, sell, license, alter, modify or otherwise use any portion of our Services except... through our interfaces and according to our Documentation."

**법적 위험:**
1. **계정 영구 차단**: Meta는 비공식 API 사용 계정을 영구 차단할 수 있습니다.
2. **법적 조치 가능성**: Meta가 법적 조치를 취할 수 있습니다.
3. **서비스 중단**: 언제든지 연동이 차단될 수 있습니다.

**본 소프트웨어 개발자 및 OpenClaw 프로젝트는 WhatsApp 연동 사용으로 인한 어떠한 결과에 대해서도 책임지지 않습니다.**

### 7.2 AI API 비용 관련

본 소프트웨어를 통해 사용하는 AI 서비스(Anthropic Claude, OpenAI GPT 등)의 API 비용은 **전적으로 사용자 책임**입니다.

**비용 발생 주의사항:**
1. `dmPolicy: "open"` 설정 시 누구나 봇에게 메시지를 보낼 수 있어 예상치 못한 비용이 발생할 수 있습니다.
2. `groupPolicy: "open"` 설정 시 모든 그룹 메시지가 AI로 전송되어 비용이 폭증할 수 있습니다.
3. 악의적 사용자의 대량 메시지 공격으로 API 비용이 급증할 수 있습니다.

**권장 사항:**
- AI 서비스 대시보드에서 월간 사용량 제한 설정
- `dmPolicy: "pairing"` 또는 `"allowlist"` 사용
- `groupPolicy: "allowlist"` 사용

## 제8조 (면책조항)

### 8.1 소프트웨어 "있는 그대로" 제공

본 소프트웨어는 **"있는 그대로(AS IS)"** 제공되며, 명시적이든 묵시적이든 어떠한 종류의 보증도 없습니다. 여기에는 상품성, 특정 목적에의 적합성, 비침해성에 대한 보증이 포함되지만 이에 국한되지 않습니다.

### 8.2 손해배상 책임 제한

어떠한 경우에도 본 소프트웨어의 개발자, 기여자, 관련 당사자는 본 소프트웨어의 사용 또는 사용 불가로 인해 발생하는 직접적, 간접적, 부수적, 특별, 징벌적 또는 결과적 손해(영업 이익 손실, 영업 중단, 영업 정보 손실 또는 기타 금전적 손실 포함)에 대해 책임지지 않습니다.

### 8.3 AI 행동에 대한 면책

AI 모델의 판단, 응답, 행동에 대해 본 소프트웨어 개발자는 어떠한 책임도 지지 않습니다. AI가 수행한 작업의 결과는 전적으로 사용자 책임입니다.

### 8.4 제3자 서비스에 대한 면책

본 소프트웨어가 연동하는 제3자 서비스(AI 제공업체, 메신저 플랫폼, 웹사이트 등)의 이용약관 위반, 서비스 중단, 계정 제재 등에 대해 본 소프트웨어 개발자는 책임지지 않습니다.

### 8.5 보안 침해에 대한 면책

본 소프트웨어의 알려진 보안 취약점(제5조에 명시)을 악용한 공격으로 인한 피해에 대해 본 소프트웨어 개발자는 책임지지 않습니다. 사용자는 이러한 위험을 인지하고 소프트웨어를 사용해야 합니다.

## 제9조 (사용자 의무)

사용자는 다음 의무를 준수해야 합니다:

1. **법률 준수**: 대한민국 법률 및 관련 국제법을 준수할 것
2. **이용약관 준수**: 연동하는 제3자 서비스의 이용약관을 준수할 것
3. **보안 관리**: API 키 및 인증 정보를 안전하게 관리할 것
4. **권한 확인**: 그룹 채팅 등에서 제3자 데이터 처리 시 적절한 권한을 확보할 것
5. **비용 관리**: AI API 사용 비용을 모니터링하고 관리할 것

이 약관의 해석 및 적용에 관하여는 대한민국 법률을 적용합니다. 본 소프트웨어 사용과 관련하여 발생하는 모든 분쟁은 대한민국 법원을 관할 법원으로 합니다.
